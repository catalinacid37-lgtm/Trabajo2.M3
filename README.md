# trabajo 2. M3
Este proyecto corresponde a un ejercicio práctico enfocado en el uso de JavaScript,
el DOM y el objeto Math, con el objetivo de reforzar la comprensión de 
operaciones matemáticas básicas y su visualización en una página web.
El ejercicio consiste en solicitar al usuario el diámetro de un círculo, calcular su área
, mostrar el resultado en pantalla y registrar automáticamente la
fecha y hora de la operación. Además, se aplican estilos utilizando 
css y también mediante JavaScript.

# utilizado:
- Html5
- Css
- JavaScript

# utilizadades
- Ingreso del diámetro por medio de un prompt.
- Cálculo automático del radio y el área.
- Visualización del resultado con dos decimales.
- Registro automático de fecha y hora.
- Aplicación de estilos dinámicos.

# estructura
-assets
-app.js
-index.html
-style.css
-README.md

# ejecución
1. Descargar o clonar el repositorio.
2. Abrir el archivo index.html en el navegador.
3. Ingresar el diámetro solicitado.
4. Visualizar el resultado en pantalla
